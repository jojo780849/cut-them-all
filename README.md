- 三把主武器：匕首(45°/55px) / 长剑(110°/85px) / 巨斧(150°/110px)
- 武器专属升级树（每武器6种专属升级）
- 打击感系统：顿帧/击退/屏幕震动/击杀粒子/伤害数字弹跳
- Boss 战：炎魔领主，三阶段四种攻击
- 浮动摇杆：任意位置触发，非线性响应
- 无敌帧系统：0.75秒无敌+方向击退
- 弧形血条跟随玩家 + 顶部精简UI
- 回血道具掉落（5%概率+磁铁吸引）
